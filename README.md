# CIDPPLUS
防灾科技学院移动教务助手

## 说明
CIDPPLUS是一款微信小程序，又名CIDP+，致力于为防灾科技学院学生打造更便捷的个人教务信息查询方式。

## 隐私
开发者严格遵守相关法律，尊重并保护您的个人隐私，所以小程序不会在服务器上保存任何个人信息，你的信息将会缓存在本地且不会与服务器有任何的交互，并开放相关设置，你可以随时清除本地缓存。

## 感谢
此小程序部分参考WeCQU、MYCIDP等开源项目的思路，并使用Vant-WeApp前端组件对界面进行优化。
